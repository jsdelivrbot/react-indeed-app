## Installing 
```
git clone https://github.com/mrdck/React-Webpack2-Babel-Starter-Files.git
cd React-Webpack2-Babel-Starter-Files
npm install
```

## Usage

To start server all what you need is to do :
```
npm start 
```
OR

```
npm run start
```

## Features

This starter pack contains :

* React
* Babel
* SASS/SCSS
